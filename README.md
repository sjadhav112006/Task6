# Task6
Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregations.

## Tools Used
- MySQL
- MySQL Workbench

## What I Did
- Created an `online_sales` table with order data
- Analyzed monthly revenue using SUM()
- Counted order volume using COUNT(DISTINCT order_id)
- Found top 3 months by sales using ORDER BY + LIMIT
